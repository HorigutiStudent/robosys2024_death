 今週の学食メニュー表を入手するコマンド.
## テスト済み環境
- Linux OS
    - Ubuntu 22.04
- Python 
    - version: 3.7 ~ 3.10
## Requirements
- Pillow
  apt でインストール(推奨)
  ```sh
  apt install python3-pil  python3-pil.imagetk
  ```
  pipでインストール(テスト済みでないため非推奨)
  ```sh
  pip3 install PIL
  ```
